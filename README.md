# ADP Challenge

## Quick Start Guide
1. Clone repo
```
$ git clone https://github.com/vannucci/ADPChallenge.git
```
2. Install
```
$ npm install
```
3. Start the script
```
$ npm start
```

## Testing Guide
1. Run mocha
```
$ mocha
```
